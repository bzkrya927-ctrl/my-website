
# موقع عبد الرحمن زكريا يحيى - محاسب مبتدئ

## نظرة عامة
هذا موقع شخصي احترافي لعبد الرحمن زكريا يحيى، مصمم لعرض مهاراته وخبراته في مجال المحاسبة.

## هيكل الملفات

```
zakaria_website/
├── index.html          # الصفحة الرئيسية
├── script.js           # ملف الجافاسكريبت
├── config.json         # ملف الإعدادات والمحتوى
├── README.md           # ملف التوثيق
└── css/                # مجلد ملفات التنسيقات
    ├── variables.css   # متغيرات الألوان والتنسيقات
    ├── base.css        # التنسيقات الأساسية
    ├── header.css      # تنسيقات الشريط العلوي
    ├── buttons.css     # تنسيقات الأزرار
    ├── hero.css        # تنسيقات قسم الهيرو
    ├── skills.css      # تنسيقات قسم المهارات
    ├── contact.css     # تنسيقات قسم التواصل
    ├── footer.css      # تنسيقات الشريط السفلي
    └── responsive.css  # تنسيقات التجاوب
```

## التحسينات التي تمت

### 1. إعادة تنظيم ملفات CSS
- تقسيم ملف CSS الكبير إلى ملفات متعددة حسب الوظيفة
- إنشاء مجلد `css` لتنظيم ملفات التنسيقات
- تحسين إمكانية الصيانة والتطوير

### 2. تحسين نظام الألوان
- استخدام درجات الأخضر بدلاً من الأزرق لتناسب مجال المحاسبة
- جعل الألوان متناسقة بين الوضع المظلم والوضع الفاتح
- إضافة متغيرات للألوان الأساسية والثانوية

### 3. تحسين الشريط السفلي (Footer)
- إضافة حدود علوية ملونة
- تحسين تصميم روابط التواصل الاجتماعي
- إضافة تأثيرات انتقالية سلسة

### 4. تحسين قسم المهارات
- إضافة حدود جانبية ملونة
- تحسين تأثيرات التحويم
- تحسين تصميم بطاقات المهارات

### 5. تحسين قسم التواصل
- تحسين تصميم نموذج الاتصال
- إضافة أيقونات للمعلومات
- تحسين تصميم روابط التواصل الاجتماعي

### 6. تحسين التجاوب
- دعم أفضل للشاشات المختلفة
- تحسين التخطيط على الأجهزة المحمولة
- تحسين أحجام النصوص والعناصر

## الميزات
- دعم اللغتين العربية والإنجليزية
- دعم الوضع المظلم والوضع الفاتح
- تصميم متجاوب يعمل على جميع الأجهزة
- رسوم متحركة سلسة باستخدام AOS
- ألوان احترافية تناسب مجال المحاسبة
- تنظيم أفضل للملفات

## التثبيت والتشغيل
1. قم بتحميل جميع الملفات
2. تأكد من وجود مجلد `css` مع جميع ملفات التنسيقات
3. افتح ملف `index.html` في المتصفح
4. استمتع بالموقع!

## التواصل
للتواصل مع عبد الرحمن زكريا يحيى، يرجى استخدام نموذج الاتصال في الموقع أو روابط التواصل الاجتماعي في الشريط السفلي.
